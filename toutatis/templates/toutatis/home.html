Toutatis

Hello, Toutatis is a simple webapp that convert the Votes of the European Parliament data taken from parltrack in the "representatives-votes" format.

If you aren't interested in those data and/or aren't using a lib/application that is implementing the "representatives" format, this website won't have a lot of interest for you.

API

If you want more fingrain access to the dumps and their meta data, you can access all of the via a tastypie API.

Warning: those data is quite big so expect some slowdown if you try to open them in your browser.

The API is accessible at this url: http://{{ request.META.SERVER_NAME }}/api/exported_revision/?format=json

Sample usage: curl http://{{ request.META.SERVER_NAME }}/api/exported_revision/?format=json | python -m json.tool
