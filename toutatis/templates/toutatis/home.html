Toutatis

Hello, Toutatis is a simple webapp that convert the Votes of the European Parliament data taken from parltrack in the "representatives-votes" format.

If you aren't interested in those data and/or aren't using a lib/application that is implementing the "representatives" format, this website won't have a lot of interest for you.

API

If you want more fingrain access to the dumps and their meta data, you can access all of the via a tastypie API.

The API is accessible at this url: http://{{ request.META.HTTP_HOST }}/api/v1/?format=json

Sample usage

Get dossier list
http://{{ request.META.HTTP_HOST }}/api/v1/dossiers/?format=json
Search dossier
http://{{ request.META.HTTP_HOST }}/api/v1/dossiers/?format=json&title__icontains=acta

You can use django field lookup, on fields : title, reference

Get a dossier
http://{{ request.META.HTTP_HOST }}/api/v1/dossiers/2925?format=json
Get proposal list

http://{{ request.META.HTTP_HOST }}/api/v1/proposals/?format=json
You can filter results like dossiers, on fields : title, reference, datetime, kind

Get a proposal
http://{{ request.META.HTTP_HOST }}/api/v1/proposals/10647/?format=json
